# 📊 Executive Sales & Marketing Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing sales, marketing performance, customer conversion, and retention to provide actionable business insights through data analytics and visualization.

The project was developed as an end-to-end business analytics solution, including data cleaning, exploratory analysis, KPI development, and dashboard creation.

---

## 🎯 Business Objectives

- Evaluate sales performance and profitability.
- Measure marketing investment effectiveness.
- Analyze customer conversion through the sales funnel.
- Study customer retention using cohort analysis.
- Build an executive dashboard to support strategic decision-making.

---

- SQL

- Pandas
- Matplotlib
- Jupyter Notebook

## 📂 Dataset Description

The project uses three datasets:

### Orders Dataset
Contains transactional information such as:

- Order ID
- Product
- Quantity
- Unit Price
- Revenue
- Customer Information
- Order Date

### Product Catalog Dataset
Contains:

- Product Category
- Product Cost
- Product Details

### Marketing Dataset
Contains:

- Campaign ID
- Marketing Channel
- Marketing Spend
- Campaign Date

## 🧹 Data Cleaning Process

The following data quality checks were performed:

- Missing value analysis
- Duplicate records validation
- Invalid numeric values detection
- Data type conversions
- Country name standardization
- Recovery of missing marketing channel information
- Dataset integration and validation
